# Smart Contracts

This module contains Solidity smart contracts for Hedera EVM.

- PropertyNFT
- LeaseAgreement
- Escrow & Rent
- SavingsVault
- FIND Token
- Reputation

## Getting Started
1. Install dependencies (see package.json or foundry/hardhat config).
2. Compile contracts.
3. Deploy to Hedera testnet.
